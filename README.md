## 💬 왜 개발하게 되었나요?

주말에 동생과 얘기 중 `나 이런거 필요한데 너 이런거 못 짜지?` 라는 말을 듣고 `아닌데? 짤 수 있는데?` 하면서 싸우다가 만들게 되었습니다.

> 단순히 강조선이랑 블러효과만 주면 간단하게 될 줄 알았는데 안 먹혀서 쉐이더 webgl 코드를 얻어와서 썼습니다. 감사합니다.
